Adding a Printer From the New Server (print01).
Number:KB:00023
URL:https://supportcenter.clayton.k12.ga.us/userui/advisory_detail?ID=23
Importance:High
Category:Printers
Platform:Windows


Step One: Open a Run prompt by striking the Windows flag key + R. It can also be opened in the start menu by searching for Run.



Step Two: Type \\print01



Step Three: Search for the correct printer in the window that appears. Note that the name of the printer no longer includes the IP address. You must know which printer is which based on it's number which will correspond with its geographical location. Double click the printer and the installation will begin.



Step Four: Verify the printer has been installed by clicking Start and devices and printers and checking if the printer name appears under printers and faxes.



Special Note: If you know the exact name of the printer, you can supply it after the \\print01. For example you can type "\\print01\Adamson Middle - ESOL" and the printer will install. It will not work without the quotation marks as there are spaces in the name of the printer. This string could even be copied into a text document and made into a .bat file and used to perform repeated installs without the need to retype the printer name.
